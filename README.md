# Sport Field Management System

A web-based platform for managing and booking sport fields.  
The system helps customers search and book available fields online, while field owners can manage their fields, schedules, bookings, and business operations from a dedicated dashboard.

## Features

### Customer
- Browse and search sport fields
- View field details, pricing, location, and availability
- Register and log in
- Book available time slots
- Track booking history and booking status

### Field Owner
- Manage owned fields
- Create, update, and maintain field information
- Manage operating hours and availability
- View and process customer bookings
- Track basic booking and revenue information

### Admin
- Manage users
- Manage field owners
- Review and manage fields across the system
- Monitor bookings on the platform
- View system-level dashboard data
